# Job Application Tracker

A modern job application tracking dashboard built with React and TypeScript.

## Features

- Dashboard analytics
- CRUD job management
- Search, filtering and sorting
- Kanban board workflow
- Protected routes
- Mock authentication
- localStorage persistence

## Tech Stack

- React
- TypeScript
- Tailwind CSS
- React Router
- Vite
